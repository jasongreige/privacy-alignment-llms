# Privacy Alignment for Open-Source LLMs

Code for my Master's thesis on reducing sensitive-data leakage in locally deployed open-source language models using PPO and DPO alignment.

## Structure

- ppo/: PPO-based privacy alignment experiments.
- dpo/: DPO-based privacy alignment experiments across healthcare, synthetic PII, criminal justice, and student-record datasets.

## Notes

Model checkpoints and Hugging Face tokens are not included in this repository.
Datasets included here are used for experimental/research purposes.

## Thesis

A Reinforcement Learning Framework for Reducing Sensitive-Data Leakage in Locally Deployed Open-Source Large Language Models.
